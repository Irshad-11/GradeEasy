📘 GradeEasy – CGPA Calculator Web Application Documentation

"A simple, powerful, and customizable CGPA calculator for students!"

🚀 1. Introduction

What is GradeEasy?

GradeEasy is a user-friendly and customizable CGPA calculator designed for university students. It helps students track their academic performance efficiently by calculating CGPA based on three grading components:

✅ Continuous Assessment Marks
✅ Mid-Term Exam Marks
✅ Semester Exam Marks

Since grading systems vary across universities, GradeEasy includes a settings panel where students can modify grading rules according to their institution’s system.

🔹 No frameworks or libraries required! This app is built using pure HTML, CSS, and JavaScript (Optional: Tailwind CSS).
🔹 Data is saved in LocalStorage, so students can access past CGPA records without signing in!

🎯 2. Project Overview

📌 Core Features

💡 Calculate CGPA based on three marking components and course credit hours.
💡 Save CGPA records to LocalStorage for quick access.
💡 Organize CGPA calculations by academic terms (e.g., 1st Year, 1st Semester).
💡 Customizable settings for different grading systems (modify grading scale, credit weightage, etc.).
💡 User-friendly UI for smooth navigation and interactions.

📝 6. CGPA Calculation Formula

GradeEasy calculates CGPA using the following formula:

CGPA=∑(Course Grade×Credit Hour)∑(Credit Hour) CGPA = \frac{\sum (Course \, Grade \times Credit \, Hour)}{\sum (Credit \, Hour)} CGPA=∑(CreditHour)∑(CourseGrade×CreditHour)​

🖩 How It Works?

1️⃣ Convert total marks into grades using the grading scale.
2️⃣ Multiply each course’s grade points by its credit hour.
3️⃣ Sum all grade points × credit hours.
4️⃣ Divide the total by sum of all credit hours to get the final CGPA.

Each tab features smooth JS-powered transitions and state synchronization, ensuring that UI elements remain visually consistent as users navigate.
